Network Admin dan IT Support

Pelatihan ini akan mempelajari pengenalan alamat IP, melakukan subnetting jaringan yang baik dan benar dan membuat suatu jaringan LAN yang saling terhubung antar perangkat. dan dapat mengidentifikasi alamat IP, melakukan subnetting jaringan yang baik dan benar, membuat suatu jaringan LAN yang saling terhubung antar perangkat dengan mampu mencapai minimum scoring penilaian 60 mengenai kemampuan membuat static routing pada saat pengumpulan unjuk keterampilan

9 Materi Pelatihan

  • Pendahuluan
  • Networking Fundamental
  • IPv4 Addressing
  • IPv6 Addressing
  • Basic Configuration Cisco
  • Wireless LANs
  • Post Test
  • Unjuk Keterampilan

Aspek Kompetensi: Pengetahuan (Knowledge)

  • Senin / 19.00 - 22.00 : Konsep dasar materi jaringan komputer
  • Selasa / 19.00 - 22.00 : Computer Network Introduction & Pengenalan IP
  • Rabu / 19.00 - 22.00 : Basic & Advanced subnetting, Static Routing
  • Kamis / 19.00 - 22.00 : VLAN (Virtual Local Area Network), Dynamic Routing
  • Jum'at / 19.00 - 22.00 : Access List, WLAN (Wireless Local Area Network)Dynamic routing

Aspek Kompetensi: Keterampilan (Skill)

  • Menentukan alamat IP
  • Merancang jaringan komputer
  • Membuat variasi dan menjaga konektivitas jaringan LAN di lingkup perusahaan
  • Mengkonstruksi kemudian mengukur dan menentukan jarak IP yang akan digunakan

VLAN (Virtual Local Area Network)

Kamu adalah Seorang network admin di suatu perusahaan, dan ingin membuat jaringan dengan menggunakan konsep VLAN untuk masing masing divisi agar bisa terhubung
 

Dynamic Routing

Access List dan WLAN (Wireless Local Area Network)

Kamu adalah Seorang network admin di suatu perusahaan, dan ingin membuat jaringan dengan menggunakan konsep Standart Access List yang mana PC 1 terhubung dengan server namun PC 2 tidak



 

Unjuk Keterampilan

Tugas Anda:
1. Assign Seluruh IP setiap device berikut dengan Subnet Mask dan Gateway
2. Aktifkan Interface pada setiap Router berikut dengan IP dan Subnet Mask
3. Aktifkan Serial pada setiap Router berikut dengan IP dan Subnet Mask
4. Lakukan Static Routing, agar semua komputer terhubung 1 dengan yang lain
